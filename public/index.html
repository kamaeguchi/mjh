Business production

  • CTI/CMS

    コールセンター向けCTI/CRMシステムを構築しました。
    バックエンド、サーバーサイドそれぞれ1名の少人数チームで開発が開始され、 初期段階でフロントエンド担当として加わり、要件定義・仕様書・設計書・実装等に携わりました。
    調査の結果、ユーザビリティに優れた競合システムがあまり無かっため、 下記のように、ユーザー目線で直感的に使用できるUI設計に注力しました。

    ・荷電画面で記録する項目をドラッグで自由にレイアウト可能
    ・荷電画面の入力フォームは値の更新時に非同期で自動保存
    ・実際のオフィスに合わせて座席レイアウトをドラッグで設定可能
    ・設定した仮想オフィス画面でステータス等がリアルタイムで確認可能

    使用言語・CMS・環境など

    FuelPHP, html/css(sass), javascript, AWS, Git, GCP, Docker

  • ECサイト

    メーカー直販(D2C)のECサイトを構築しました。 CMSには当時EC系国内シェアトップのEC-CUBEではなく、世界的にシェアが伸びていたMagento2を選択し、 ニアショア・オフショア含め、フロント開発約50名・システム開発10名の体制で約1年弱かけて製作。 フロントサイドのPLとして、要件定義・スケジュール管理・アサイン調整・進捗管理などを行いました。 ニア/オフショアの品質管理のため、コンポーネントと製作ルールの共有を徹底したため、 トラブル無く無事リリースすることができました。

    使用言語・CMS・環境など

    Magento2, html/css(sass), javascript, Git, XAMPP

  • 国内航空会社予約サイト

    国内航空会社の予約サイトを構築しました。
    社内の別チームが製作していましたが、問題があったため再製作の指揮をとりました。
    汎用的な機能やコンポーネントをまとめ直し、無事リリースが完了しました。

    使用言語・CMS・環境など

    html/css, javascript, Git, XAMPP

  • Googleしごと検索スクレイピングツール

    Googleしごと検索で検索した結果をCSV出力するツールを製作しました。 検索結果は検索を実行したデバイスの位置情報を使用するため、 位置情報を上書きして任意の地点でのキーワードの検索結果を表示します。

    使用言語・CMS・環境など

    AWS Lambda, Node.js, chromium, puppeteer, GCP MAP API, GCP Geocoding API
